---
title: "Integrating"
---

Put your extracted document data to work with integrations. Sensible offers the following integration options for automating extractions: 

| method                                 | getting started                                 | description                                                                                                                                           |
| -------------------------------------- | ----------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| SDKs                                   | [SDK guides](https://sensible-87.mintlify.app/integrations/sdk-guides)                  | Extract and classify documents.                                                                                                                       |
| [Zapier](https://sensible-87.mintlify.app/integrations/zapier-overview/overview)                 | [Zapier tutorial](https://sensible-87.mintlify.app/integrations/zapier-overview/zapier-tutorial) | Transform data in PDFs and other document file types into emails, databases, Google sheets, and other Zapier-supported destinations.                  |
| [API](https://sensible-87.mintlify.app/api-reference/introduction) | [API quickstart](https://sensible-87.mintlify.app/integrations/api-quickstart)              | Extract and classify documents and manage extraction configuration.                                                                                   |
| Quick extraction                       | [Quick extraction](https://sensible-87.mintlify.app/integrations/quick-extraction/overview)      | Drag and drop documents to extract data from documents in the Sensible app. Upload files in bulk, and download extracted data as JSON or Excel files. |
